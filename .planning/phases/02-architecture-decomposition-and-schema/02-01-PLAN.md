Create `src/ar/core/SceneManager.ts` as a class following the research pattern from 02-RESEARCH.md "SceneManager Extract" section. Key behaviors:

- Constructor takes `canvas: HTMLCanvasElement`. Creates Scene, PerspectiveCamera (FOV 63, display aspect, near 0.1, far 1000, position.z = 2), WebGLRenderer (alpha: true, clearColor 0x000000/0).
- **PERF-01**: Cap DPR at 1.0 on mobile. Detection: `window.innerWidth < 768 || (navigator.maxTouchPoints > 0 && screen.width < 1024)`. Otherwise `Math.min(window.devicePixelRatio, 2)`.
- Add AmbientLight (0xffffff, 0.6) and DirectionalLight (0xffffff, 0.8, position 0,1,1).
- **PERF-02**: `dirty: boolean` flag. `renderIfDirty()` method only calls `renderer.render()` when dirty is true, then resets it.
- `get visibleDims()` getter: computes visible width/height at camera z-distance from FOV.
- `swapModel(newModel: THREE.Object3D | null)`: removes old model (disposes geometry + materials), adds new model, caches material references from new model.
- **PERF-03**: `cachedMaterials: THREE.Material[]` populated in swapModel. `updateOpacity(opacity: number)` iterates cached array without traversal.
- `handleResize()`: updates renderer size, camera aspect, projection matrix, marks dirty.
- `dispose()`: calls swapModel(null), disposes renderer.
- All properties that ARExperience needs (renderer, scene, camera) are public for direct ref access.

Create `src/ar/core/ModelLoader.ts`:

- Export `ModelResult` interface: `{ wrapper: THREE.Group; dims: { w: number; h: number; d: number } }`.
- Export `async function loadModel(url: string): Promise<ModelResult>` -- uses GLTFLoader to load the URL, computes bounding box via Box3.setFromObject, centers model at origin (position.sub(center)), extracts dims from size, wraps in a Group (visible: false), returns wrapper + dims.
- This is extracted from ARExperience.tsx lines 198-234.
- Do NOT handle scene.add or opacity in this module -- that is SceneManager.swapModel's job.

Import Three.js as `import * as THREE from 'three'` and GLTFLoader from `'three/examples/jsm/loaders/GLTFLoader.js'` -- matching existing import paths in ARExperience.tsx.
