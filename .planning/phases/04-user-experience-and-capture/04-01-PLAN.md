- For all other states, return the same generic messages as current TrackingGuidance (delegate to caller -- this function only handles waiting_for_pose and partial_tracking).

- For each landmark in garmentPreset.requiredLandmarks + optionalLandmarks, check if visibility[index] < visibilityThreshold.
- Map missing landmark indices to human-readable body part names using a local BODY_PART_NAMES map:

- Deduplicate (left shoulder + right shoulder = just "shoulders").
- Return array of missing part names, e.g. ["hips", "ankles"].
